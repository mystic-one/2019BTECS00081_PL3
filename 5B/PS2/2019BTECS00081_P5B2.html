Program to find Sum and Average

Problem Statement | Code | Sample Input | Sample Output | Variables Used | Complexity

Problem Statement:

Write a program in any language to read 10 numbers from keyboard and find their sum and average.

Code

		
#include <iostream>
using namespace std;

int main() {
	cout << "Sum And Average Calculator\n" << endl;

	int total_numbers = 10;
	int sum = 0;
	float average = 0;

	int numbers[10];

	for(int i = 0; i < total_numbers; i++){
		cout << "Enter Number " << i+1 << ": ";
		cin >> numbers[i];
	}

	for(int i = 0; i < total_numbers; i++){
		sum += numbers[i];
	}

	average = sum/total_numbers;

	cout << "\nThe sum of the numbers is " << sum << endl;
	cout << "\nThe average of the numbers is " << average << endl;

	return 0;
}

Complexity

Time Complexity:

The program runs in linear time. Therefore, the time complexity is : O(n)

Space Complexity:

The program uses space equivalent to total numbers required to be computed. Hence the space complexity is : O(n)
